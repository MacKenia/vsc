# Github Desktop
这是由GitHub开发的，用于替代git的客户端，它可以替代git完成大部分工作，总所周知git是纯命令操作的代码系统，而GitHub Desktop就不一样了，它是由GUI界面的而且美观

# 代码仓库
代码仓库即储存代码的地方，它类似于网盘，但是它更偏向储存代码，而不是各种媒体，对代码的管理，共享等
